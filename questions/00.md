# Multiple Choice Question

What is the correct syntax of a `while` loop?

## Incorrect

```java
while (initializing-statement; conditional; update-statement) {
    // Statements to run through each iteration of the loop
}
```

## Correct

```java
while (conditional) {
    // Statements to run through each iteration of the loop
}
```

## Incorrect

```java
while {
    // Statements to run through each iteration of the loop
}
```

## Incorrect

```java
while {
    // Statements to run through each iteration of the loop
} (conditional);
```
