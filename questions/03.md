# Multiple Choice Question

What is the correct syntax of a `do-while` loop?

## Incorrect

```java
do-while (initializing-statement; conditional; update-statement) {
    // Statements to run through each iteration of the loop
}
```

## Incorrect

```java
do (conditional) {
    // Statements to run through each iteration of the loop
} while;
```

## Incorrect

```java
do-while {
    // Statements to run through each iteration of the loop
}
```

## Correct

```java
do {
    // Statements to run through each iteration of the loop
} while (conditional);
```
